윈도우 터미널창 프로젝트 루트 리렉토리 시작
cd front
npm run dev
터미널창에 뜬 링크로 항해

q 엔터로 종료


admin(퀴즈 준비)
메인 이미지: 모든 문제의 정답 공개전 출력되는 이미지입니다.
문제 질문: 문제의 질문을 넣는 칸입니다.
문제 정답: 문제의 정답을 넣는 칸입니다.
파일 선택: 각 문제의 정답 공개 후 메인 이미지에서 해당 이미지로 변경됩니다.
관련 링크: 문제와 관련된 링크가 있다면, 해당 링크 기입시 정답 공개 후 밑에 출력됩니다


guest(실제 퀴즈 진행)
예시문제: 3명 기준으로 진행된 샘플 문제를 진행합니다. 
퀴즈설명: 룰 설명이 진행된 유튜브 링크로 항해합니다. 만약 실제 퀴즈 진행 방식까지 보여주고 싶으면 해당 영상에서 3분 12초쯤으로 이동하시면 됩니다
문제선택: 1번부터 25번까지의 문제를 고릅니다. 

문제창
시작: 문제가 한글자씩 출력됩니다.
중단: 문제의 출력이 중단됩니다
다음: 출력된 문제를 지웁니다
정답: 문제, 답을 공개하고 이미지를 변경한 다음 링크를 출력합니다
